Tracking real data flowing through the three critical workflows:

### Workflow 1: Folder Skip
```
scan_results (dict from DuplicateClassifier.classify())
  → show_folder_selection() extracts files
  → get_unique_parent_folders() returns ['/path/a', '/path/b', '/path/c']
  → ui.display_folder_list() renders numbered list
  → parse_selection_input("1,3") returns [1, 3]
  → Indices mapped to paths: {'/path/a', '/path/c'}
  → NavigationContext.selected_folders_skip = {'/path/a', '/path/c'}
  → filter_candidates_by_flags() uses context to exclude matching folders ✓ DATA FLOWING
```

### Workflow 2: File Flagging
```
scan_results (dict)
  → show_file_flagging() extracts file paths
  → extract_duplicate_files() returns all paths from dict
  → Create FileRecord objects with real disk sizes
  → ui.display_file_list(files, flagged_set) renders with sizes
  → parse_selection_input("2,4") returns [2, 4]
  → Indices mapped to file paths: {'/path/file2.txt', '/path/file4.txt'}
  → NavigationContext.flagged_files = {'/path/file2.txt', '/path/file4.txt'}
  → filter_candidates_by_flags() uses context to exclude flagged files ✓ DATA FLOWING
```

### Workflow 3: Back Navigation State Preservation
```
Initial: NavigationContext(scan_results={...}, selected_folders_skip=set(), flagged_files=set())
  → User selects option 1 (skip folders)
  → show_folder_selection() updates context.selected_folders_skip = {'/path/a'}
  → User presses 'b' to go back
  → context returned unchanged with selected_folders_skip still = {'/path/a'} ✓ DATA PRESERVED
  → User selects option 2 (flag files)
  → show_file_flagging() updates context.flagged_files = {'/path/file1.txt'}
  → Both selections preserved simultaneously ✓ DATA ACCUMULATES
```
