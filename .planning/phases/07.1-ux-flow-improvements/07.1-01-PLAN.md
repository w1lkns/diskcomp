Add prompt_confirm() helper function to diskcomp/cli.py near the top of the file (after imports, before parse_args). This function validates user input against a whitelist of valid keys.

Function signature:
```python
def prompt_confirm(message: str, valid_keys: list[str], ui: UIHandler) -> str:
```

Behavior:
1. Display message using ui.print() or input() directly
2. Call input() and strip whitespace
3. Loop until input is non-empty AND in valid_keys (case-sensitive)
4. On invalid input, call ui.error(f"Invalid choice. Enter one of: {', '.join(valid_keys)}")
5. Return the validated key (single character)

Example usage:
```python
choice = prompt_confirm("Delete this file? (y/n/skip): ", ['y', 'n', 'skip'], ui)
```

Integration point: Replace all existing raw input() calls in deletion workflows with prompt_confirm() calls. This task ONLY adds the function; integration happens in later tasks.

Docstring: Explain the validation loop, valid_keys format, and return value.
