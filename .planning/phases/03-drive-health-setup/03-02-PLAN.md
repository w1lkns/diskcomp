**Function 1: get_drives() -> list[DriveInfo]**
- Returns: List of DriveInfo objects for all mounted drives
- Logic:
  - If PSUTIL_AVAILABLE:
    - Call psutil.disk_partitions(all=False) to get list of partitions
    - For each partition, create DriveInfo with device, mountpoint, fstype
    - Call check_drive_health(mountpoint) to get space stats
    - Populate DriveInfo with total_gb, used_gb, free_gb, is_writable from health check
    - Return list of DriveInfo objects
  - Else:
    - Call _get_drives_subprocess() to enumerate drives
    - Return result

**Function 2: _get_drives_subprocess() -> list[DriveInfo]**
- Platform-agnostic fallback when psutil unavailable
- Detects platform and calls appropriate helper:
  - sys.platform == 'darwin': call _get_drives_macos()
  - sys.platform == 'win32': call _get_drives_windows()
  - sys.platform.startswith('linux'): call _get_drives_linux()
  - Otherwise: return empty list []

**Function 3: _get_drives_macos() -> list[DriveInfo]**
- Uses subprocess to call `diskutil list -plist` and parse output
- Extract device, mount point, fstype from plist output
- Return list of DriveInfo or empty list on error
- On error: log to stderr and return []

**Function 4: _get_drives_linux() -> list[DriveInfo]**
- Reads /etc/mtab or /proc/mounts to get mounted filesystems
- Parses lines to extract device, mountpoint, fstype
- Return list of DriveInfo or empty list on error
- On error: log to stderr and return []

**Function 5: _get_drives_windows() -> list[DriveInfo]**
- Uses subprocess to call `wmic logicaldisk get name,filesystem,size,freespace /format:csv`
- Parses CSV output to extract drive letters, filesystem, space stats
- Return list of DriveInfo or empty list on error
- On error: log to stderr and return []

**Function 6: _prompt_for_drive_index(prompt_text: str, max_index: int) -> int**
- Loops until user enters valid number
- Input: prompt_text (e.g., "Which drive to KEEP?"), max_index (number of drives)
- Returns: 0-based index of selected drive
- Logic:
  - While True:
    - Call input(f"{prompt_text} (1-{max_index}): ").strip()
    - Try to convert to int
    - If 1 <= int_value <= max_index:
      - Return int_value - 1 (convert to 0-based index)
    - Else:
      - Print "Invalid selection. Enter a number between 1 and {max_index}."
      - Loop again
    - On ValueError (not a number):
      - Print "Invalid input. Please enter a number."
      - Loop again

**Function 7: interactive_drive_picker() -> Optional[dict]**
- Returns: {'keep': '/path/to/keep', 'other': '/path/to/other'} or None on error
- Logic:
  - Call get_drives() to enumerate all mounted drives
  - If no drives:
    - Print "Error: No mounted drives detected." to stderr
    - Return None

  - Print header "=== Available Drives ===\n"
  - For each drive (enumerated 1-based):
    - Print "{i}. {mountpoint}"
    - Print "   Device: {device}"
    - Print "   Filesystem: {fstype}"
    - Print "   Space: {used_gb}GB / {total_gb}GB used"
    - Print "   Writable: {'Yes' if is_writable else 'NO (READ-ONLY)'}"
    - If drive has warnings (from check_drive_health):
      - For each warning: print "   WARNING: {warning}"
    - Print blank line

  - Call _prompt_for_drive_index("Which drive to KEEP?", len(drives))
  - Store keep_drive = drives[keep_index]['mountpoint']

  - Print f"\nKeep drive: {keep_drive}\n"

  - Build remaining list (all drives except keep_index)
  - If remaining is empty:
    - Print "Error: Need at least 2 drives to compare." to stderr
    - Return None

  - Print "Remaining drives for comparison:"
  - For each remaining drive (enumerated 1-based):
    - Print "{i}. {mountpoint}"

  - Call _prompt_for_drive_index("Which drive to compare (OTHER)?", len(remaining))
  - Store other_drive = remaining[other_index]['mountpoint']

  - Print f"\nSelected: Keep={keep_drive}, Other={other_drive}"

  - Return {'keep': keep_drive, 'other': other_drive}
