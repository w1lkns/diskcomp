Create diskcomp/scanner.py with a FileScanner class and noise pattern definitions. This module walks filesystems recursively, skips OS noise, and collects file metadata.

1. **NOISE_PATTERNS dict** — define noise patterns per platform:
   ```python
   NOISE_PATTERNS = {
       'all': ['.DS_Store', '.localized', '.Spotlight-V100', '.TemporaryItems', '.Trashes'],
       'windows': ['Thumbs.db', '$RECYCLE.BIN', '$Recycle.Bin', 'System Volume Information', '$SYSTEM.SAV', 'NTFS.log', 'pagefile.sys', 'hiberfil.sys'],
       'linux': ['.cache', '.config', '.local', '.mozilla', '.thumbnails'],
   }
   ```
   Include a function `should_skip_file(name: str, platform: str) -> bool` that checks if a filename/dirname should be skipped.

2. **FileScanner class** with methods:
   - `__init__(self, root_path: str, min_size_bytes: int = 1024)` — initialize scanner for a drive path, set minimum file size to 1KB
   - `scan(self, dry_run: bool = False, limit: int = None) -> ScanResult` — main scanning method:
     * Walk root_path recursively using os.walk()
     * For each file, check: (a) is it noise? (b) is it >= min_size_bytes? If yes to both, add to files
     * Track file count, total size, noise count, and any errors
     * If dry_run=True, count files without opening them (fast path) — just use os.path.getsize()
     * If limit is set, stop after collecting limit files (for testing)
     * On error reading a file (PermissionError, OSError), log to errors list and continue (don't crash)
     * Return ScanResult with all collected data

   - `_get_relative_path(self, abs_path: str) -> str` — convert absolute path to relative path for reporting
   - `_is_noise(self, name: str) -> bool` — check if filename/dirname matches noise patterns

3. **Platform detection**:
   - Use sys.platform to detect OS: 'darwin' (macOS), 'win32' (Windows), 'linux' (Linux)
   - Pass platform to noise filter function

4. **Error handling**:
   - Catch PermissionError, OSError, etc. during directory walk and file size checks
   - Log errors to ScanResult.errors list with {path, reason}
   - Never crash — always return partial ScanResult with errors recorded

5. **Imports**:
   - from diskcomp.types import FileRecord, ScanResult, ScanError, InvalidPathError
   - import os, sys, pathlib

Example usage (comment in file):
```python
scanner = FileScanner("/Volumes/MyDrive")
result = scanner.scan(dry_run=False, limit=None)
print(f"Found {result.file_count} files, {result.total_size_bytes} bytes, {result.skipped_noise_count} noise files skipped")
```
