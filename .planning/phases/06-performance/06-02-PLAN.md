   **CRITICAL: Benchmark must measure real hashing work, not just list iteration.** Use the following approach:

   Method A (PREFERRED — Real temp files with SHA256):
   - Create a temp directory structure with actual small files (1-10 KB each)

   - Measure wall-clock time for each approach
   - Assert: (time_single_pass / time_two_pass) >= 5.0

   Method B (Alternative — Mock hashing with measurable I/O work):
   - Create FileRecord objects with dummy paths

   - This ensures the callback's work dominates timing, not just list iteration
   - Time both approaches and assert >= 5.0 speedup

   **DO NOT use the iteration-only approach** (lines 213-221 of current pseudocode: `hashed_all = [r for r in ...]`). List comprehension without hashing work will not demonstrate the filter's benefit.

   Example with real temp files:
   ```python
   import tempfile
   import os

   # Create temp directory and files
   with tempfile.TemporaryDirectory() as tmpdir:
       # Create 200 small files (~1 KB each)
       all_files = []
       for i in range(200):
           filepath = os.path.join(tmpdir, f'file_{i}')
           with open(filepath, 'wb') as f:
               f.write(b'x' * 1024)  # 1 KB

           # Build FileRecord
           record = FileRecord(
               path=filepath,
               rel_path=f'file_{i}',
               size_bytes=1024 + (i % 100),  # Sizes 1024-1123 (100 files), 1024-1123 repeat (100 more) + unique offsets
               hash=None
           )
           all_files.append(record)

       # Distribute: keep_records = first 100, other_records = second 100
       # First 100 keep: sizes 1024-1123 (will have matches in other)
       # Second 100 other: sizes 1024-1123 (matches keep) + offset (1124+)
       keep_records = all_files[:100]
       other_records = all_files[100:]  # Adjust sizes so some overlap

       # Adjust sizes in other_records to create overlap
       for i, rec in enumerate(other_records[:100]):
           if i < 50:
               rec.size_bytes = 1024 + i  # Overlap with keep
           else:
               rec.size_bytes = 2000 + i  # Unique

       # Time single-pass: hash all 200
       start = time.time()
       hasher = FileHasher()
       hashed_all = hasher.hash_files(keep_records + other_records)
       time_single_pass = time.time() - start

       # Time two-pass: filter then hash ~100
       start = time.time()
       filtered_keep, filtered_other, stats = filter_by_size_collision(keep_records, other_records)
       hasher = FileHasher()
       hashed_filtered = hasher.hash_files(filtered_keep + filtered_other)
       time_two_pass = time.time() - start

       ratio = time_single_pass / time_two_pass
       print(f"Speedup ratio: {ratio:.1f}×")
       self.assertGreaterEqual(ratio, 5.0, f"Expected ≥5× speedup, got {ratio:.1f}×")
   ```
