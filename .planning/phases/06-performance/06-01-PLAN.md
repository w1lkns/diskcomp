<objective>
Implement the two-pass scan optimization: after collecting file sizes, filter out files with sizes that don't exist on the other drive before hashing. This provides the performance boost for large drives.

Purpose: Large drives (500GB+) complete scanning and hashing in reasonable time by skipping obviously-unique files.
Output: Integrated size filter with status line and progress bar updates. All existing tests pass with identical results.
</objective>
