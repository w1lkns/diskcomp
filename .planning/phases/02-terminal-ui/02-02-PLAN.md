Add a new method `hash_files()` to FileHasher that processes multiple file records with per-file progress callback and speed/ETA calculation:

1. **Add new method** (do NOT modify existing hash_file or hash_file_record):
   ```python
   def hash_files(
       self,
       records: List[FileRecord],
       on_file_hashed: Optional[Callable[[int, int, float, int], None]] = None,
   ) -> List[FileRecord]:
       """
       Hash multiple files with per-file progress callback.

       Args:
           records: List of FileRecord objects to hash
           on_file_hashed: Optional callback(current_index, total, speed_mbps, eta_secs)
                          called after hashing each file

       Returns:
           List of FileRecord objects with hash field populated
       """
   ```
   Import `Callable, List` from typing.

2. **Speed and ETA calculation**:
   - Track start time: `import time; start_time = time.time()`
   - After each file: `elapsed = time.time() - start_time`
   - Total bytes hashed so far: sum of all file sizes
   - Speed: `speed_mbps = (total_bytes_hashed / elapsed) / (1024 * 1024)` to get MB/s
   - Remaining bytes: sum of sizes in unprocessed records
   - ETA: `eta_secs = int(remaining_bytes / bytes_per_sec)` if speed > 0 else 0

3. **Iterate and callback**:
   ```python
   for i, record in enumerate(records):
       record = self.hash_file_record(record)  # Use existing method
       if on_file_hashed:
           # Calculate speed and ETA
           # Call callback(i + 1, len(records), speed_mbps, eta_secs)
       hashed_records.append(record)
   return hashed_records
   ```

4. **Error handling**:
   - If hash_file_record returns error (record.error is not None), still count it and continue
   - Report progress even on errors

5. **Import additions**:
   Add `from typing import Callable, List` at top of file if not present.

Note: Do NOT change hash_file or hash_file_record signatures. Only add new hash_files method.
  </action>

- ETA calculated as remaining bytes / current speed in seconds

   ```python

   keep_result = keep_scanner.scan(

       limit=args.limit,
       on_folder_done=lambda folder_path, count: ui.on_folder_done(folder_path, count)
   )
   ```

4. **Modify other scanner call** similarly:
   ```python
   ui.start_scan(args.other)
   other_result = other_scanner.scan(
       dry_run=args.dry_run,
       limit=args.limit,
       on_folder_done=lambda folder_path, count: ui.on_folder_done(folder_path, count)
   )
   ```

5. **Modify hash loop**:
   OLD:
   ```python
   keep_result.files = [hasher.hash_file_record(f) for f in keep_result.files]
   other_result.files = [hasher.hash_file_record(f) for f in other_result.files]
   ```
   NEW:
   ```python
   ui.start_hash(len(keep_result.files) + len(other_result.files))
   all_records = keep_result.files + other_result.files

   def on_file_hashed_callback(current, total, speed_mbps, eta_secs):
       ui.on_file_hashed(current, total, speed_mbps, eta_secs)

   hashed_records = hasher.hash_files(all_records, on_file_hashed_callback)

   # Split back into keep/other
   keep_count = len(keep_result.files)
   keep_result.files = hashed_records[:keep_count]
   other_result.files = hashed_records[keep_count:]
   ```
